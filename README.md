### üßæ Steps Performed

   - Signed up at [https://protonvpn.com](https://protonvpn.com).

2. **Install the VPN Client**
   - Downloaded and installed ProtonVPN for Windows.
   - Logged in using free account credentials.

3. **Connect to a VPN Server**
   - Connected to a server in **japan**.
   - Connection status showed **Connected** in the client.

4. **Verify IP Address Change**
   - Before connection: IP showed **India**.
   - After connection: IP changed to **japan**.
   - Confirmed VPN tunneling and encryption are active.

5. **Test Secure Browsing**
   - Visited several websites and confirmed secure (HTTPS + VPN tunnel) connections.

6. **Disconnect and Compare**
   - Disconnected from the VPN.
   - Speed slightly improved, and IP reverted to India.

7. **Research**
   - VPNs use **AES-256 encryption** and protocols like **OpenVPN/WireGuard** for security.
